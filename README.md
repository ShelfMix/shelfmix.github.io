# ShelfMix.com

ShelfMix is the collection/shelf front door for ShareShuffle.

## Files

- `index.html` — simple landing page
- `404.html` — GitHub Pages redirect handler
- `CNAME` — custom domain setting for `shelfmix.com`

## Redirect behavior

`https://shelfmix.com/bass-gear` redirects to:

`https://shareshuffle.com/shelf.html?s=bass-gear`

## GitHub Pages setup

1. Create a public repo, recommended name: `shelfmix.github.io` or `shelfmix.com`.
2. Upload these files to the repo root.
3. Turn on GitHub Pages from the `main` branch/root.
4. Add custom domain: `shelfmix.com`.
5. Point DNS to GitHub Pages.
